== Cucumber and Selenium-Webdriver Test Steps

Gem Dependencies: cucumber, selenium-webdriver, rspec

Add require 'blackboxd' to your cucumber features/support/env.rb file and start these steps found in lib/blackboxd.rb **

== Example Feature:

features/google_test.feature

       Feature: Google test
 
       Scenario: Click Signin Button
         * I am on the google home page
         * I click_on "gb_70" "id"
